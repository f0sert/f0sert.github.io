Проверка статуса порта

Введите IP/домен и порт. Сайт попытается определить статус порта: открыт / закрыт / фильтруется.
Готово к проверке
Совет: для HTTP-тестов откройте сайт по http://.
HTTPS/ WSS тесты работают из https:// контекста.
Ограничения браузера: нельзя делать "чистый TCP/UDP". Результат верен для HTTP(S)/WebSocket. Для прочих протоколов возможны ложные «закрыт/фильтруется».
Фронтенд-сканер: эвристики по тайм-аутам и типам ошибок.
Тайм-аут по умолчанию: 4000 ms
